# Huellas---MVC-PHP-Shoestore
Página web de tienda de zapatos deportivos creada con el modelo Modelo-Vista-Controlador con HTML, JavaScript y Sematic UI en el front y PHP y MySQL en el back.

## Requisitos
1. PHP 7.4+
2. MySQL 5.7+
3. Apache

## Instalación
1. Clonar repositorio
2. Importar estructura de `schema > tienda-schema.sql` (incluye creación de base de datos)
3. Configurar credenciales de config/database.php
